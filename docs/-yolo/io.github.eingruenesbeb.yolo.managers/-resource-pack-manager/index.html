ResourcePackManager

class ResourcePackManager

This is a utility class, that manages setting up a resource pack, as it is described in the plugin's config. It provides asynchronous validation of the provided resource-pack and a fallback, if no custom resource pack should be used or the custom resource pack couldn't be validated. It also provides a method to apply the resource pack to a player.
